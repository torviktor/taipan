# 🥋 Тайпан — Платформа управления спортивным клубом

> Полноценный веб-продукт для клуба тхэквондо «Тайпан» (г. Павловский Посад) —
> от публичного сайта до внутренней CRM с геймификацией, Telegram-интеграцией
> и многоролевым личным кабинетом.

**Сайт:** https://taipan-tkd.ru
**Telegram канал:** @taipan_tkd | **Бот:** @taipan_tkd_bot

---

## О проекте

Этот проект — пример того как современный спортивный клуб может работать
без сторонних CRM, таблиц Excel и групп в мессенджерах.
Всё в одном месте: администрирование, коммуникация с родителями,
учёт финансов, геймификация и аналитика.

Проект спроектирован и реализован с нуля как пет-проект
с прицелом на реальное production-использование.
Сайт работает в боевом режиме и обслуживает реальных пользователей.

### Что делает проект нестандартным

- **Геймификация** — система ачивок и рейтинга мотивирует детей
  ходить на тренировки и участвовать в соревнованиях
- **Многоролевая архитектура** — parent / manager / admin
  с тонкой настройкой прав доступа
- **Telegram-first коммуникация** — бот для личных уведомлений +
  канал для публичных анонсов, полная синхронизация с сайтом
- **Финансовый модуль** — учёт взносов с логикой бюджетников,
  переносом долгов и заморозкой прошлых периодов
- **Автоматизация** — рейтинг, ачивки, уведомления, Celery-задачи —
  всё считается и отправляется без ручного вмешательства

## Стек технологий

| Слой | Технология | Детали |
|------|-----------|--------|
| Backend | FastAPI (Python 3.11) | REST API, async, BackgroundTasks |
| Frontend | React 18 + Vite | SPA, динамический роутинг |
| БД | PostgreSQL 15 | SQLAlchemy ORM, авто-миграции |
| Очереди | Celery + Redis | Фоновые задачи, периодические задачи |
| Авторизация | JWT | Role-based access control |
| Уведомления | Telegram Bot API | Вебхук, личные + канал |
| Деплой | Docker Compose + Nginx | SSL, rate limiting, reverse proxy |
| CI/CD | GitHub Actions | Автодеплой при пуше в main |
| Новости | YandexGPT + парсинг | Автогенерация новостного контента |

## Функционал

### Публичная часть сайта
- Главная страница с hero-секцией и CTA
- «Почему Тайпан» — маркетинговая страница с геймификацией,
  живым мокапом рейтинга и реальными ачивками
- О клубе (8 разделов: история, устав, структура сезона и др.)
- Расписание тренировок
- Новостная лента
- Календарь событий (FullCalendar)
- Зал Славы с лучшими спортсменами сезона
- Форма записи на пробное занятие
- Квиз для подбора формата занятий

### Личный кабинет — родитель
- Профиль спортсмена с фото и поясом
- Посещаемость с графиком по месяцам
- Рейтинг по сезонам и возрастным категориям (6–7, 8–9, 10–11, 12–14, 15–17)
- Система ачивок с автоначислением
- Аналитика прогресса
- Уведомления с ответами (участвую / не участвую)
- Статус взносов и история долгов
- Заявки на индивидуальные занятия

### Личный кабинет — менеджер
- Всё что у родителя
- Панель управления взносами своей группы:
  бюджетники/внебюджетники, учёт оплат, долги,
  рассылка уведомлений, заморозка прошлых периодов

### Личный кабинет — тренер (admin)
- Управление спортсменами (фото, пояс, группа, архив)
- Соревнования: создание → опрос → результаты → рейтинг → ачивки
- Аттестации: создание → участники → результаты → обновление поясов
- Сборы: создание → опрос → экспорт xlsx
- Журнал посещаемости с ачивками
- Зал Славы: управление + лучшие спортсмены сезона
- Новости с фото + автопубликация в Telegram
- Стратегия клуба
- Заявки: вступление + индивидуальные тренировки
- Взносы: мониторинг всех групп (только просмотр)
- Аналитика клуба

## Telegram интеграция

Одна из ключевых особенностей проекта — полноценная двухканальная
Telegram-интеграция.

### Публичный канал @taipan_tkd
Автоматически публикуется при действиях на сайте:
- Новости клуба (текст + превью)
- Анонсы соревнований и сборов
- События из календаря

### Личный бот @taipan_tkd_bot
Привязка аккаунта: `/link 79991234567`
После привязки пользователь получает все уведомления из ЛК в Telegram:
- Приглашения на соревнования и сборы
- Уведомления об аттестациях
- Напоминания о взносах и задолженностях

Команды бота: `/start` `/stop` `/events` `/week` `/month` `/news` `/link`

Техническая реализация: вебхук на FastAPI, BackgroundTasks для
асинхронной отправки, раздельная логика для канала и личных сообщений.

## Геймификация

### Система ачивок
Автоматически начисляются через Celery и триггеры в API:

| Категория | Ачивки |
|-----------|--------|
| Посещаемость | 10, 50, 100 тренировок; 100% за месяц |
| Соревнования | Первый турнир, золото, топ-3 сезона |
| Аттестации | Повышение пояса |
| Сборы | Участие в выездных сборах |
| Верность | 1 год в клубе |

### Рейтинговая система
- Формула: `значимость × ln(очки + 1)`
- Учитывает: кол-во боёв, место, тип дисциплины
- Возрастные категории: 6–7, 8–9, 10–11, 12–14, 15–17 лет
- Сезон: сентябрь–август

## Деплой

### Переменные окружения (.env)
```env
POSTGRES_USER=
POSTGRES_PASSWORD=
POSTGRES_DB=
SECRET_KEY=
TELEGRAM_BOT_TOKEN=
TELEGRAM_CHANNEL_ID=
SITE_URL=
REDIS_URL=
ANTHROPIC_API_KEY=
YANDEX_API_KEY=
YANDEX_FOLDER_ID=
```

### Запуск
```bash
git clone https://github.com/torviktor/taipan.git /opt/taipan
cd /opt/taipan
cp .env.example .env  # заполнить переменные
docker compose up -d --build
```

### CI/CD
GitHub Actions автоматически деплоит при пуше в `main`.
Бэкенд пересобирается при изменении Python-кода.
Фронтенд пересобирается при изменении JS/JSX.

## Безопасность

- Порты 8000 и 3000 закрыты извне (127.0.0.1)
- Nginx: X-Frame-Options, XSS-Protection, Referrer-Policy, HSTS
- Rate limiting: 30 req/min на /api/
- SSL через Let's Encrypt с автообновлением
- .env в .gitignore, права 600 на сервере
- JWT с ролевой моделью доступа
